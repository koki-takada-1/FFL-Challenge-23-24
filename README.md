# FFL-Challenge-23-24
FLL2023~2024についての概要
## STEAM教育とは
STEAMとは、Science（科学）、Technology（技術）、Engineering（工学・ものづくり）、Art（芸術・リベラルアーツ）、Mathematics（数学）の5つの単語の頭文字を組み合わせた教育概念

## 予選大会
日程：2023年12月16日(土)・17日（日）

会場：板橋区立成増ケ丘小学校（予定）

## 全国大会
日程：2024年 2月1２日（祝）

会場：東京大学本郷キャンパス（予定）
